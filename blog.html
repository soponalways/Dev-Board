Question-1: What are the different ways to select an element in the DOM?

Answer:
There are severel methods available to select element using dom, Like:

  • Selecting By Id: Selects a single elemt by its unique id attribute.
  • Selecting By Class Name: Select all elements that have a specify class name. That's returns a Live HTMLCollection.
  • Selecting By Tag Name: Select the all elements that specific Tag Name
  • Selecting By CSS Selectors: Select single or all elements that match specifyed CSS Selectors that returns static NodeList.
  • Selecting By attribute: Select all elements thats select By Specified attribute. thats Returns live Node List.

    • Question-3: What is event delegation in the DOM?

    Answer:
    Event Delegation is a techinique in JavaScript whre you delegate the responsibility of handling events to a parents Elements, Rather than attching event listener to individual child elements.

    Question-4: What is event bubbling in the DOM?

    Answer:
    Event Bubbling is a mechanism in the DOM. Where an event trigered a specific element propagate upward through its ancestor elements in the DOM tree.
